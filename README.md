# 🏴‍☠️ Pirataria 2026 — Piratas UFOP

Site de vendas de ingressos para o evento **Pirataria: Bem-vindos, Marujos!**  
Associação Atlética Acadêmica de Educação Física — UFOP

## Como usar

Abra o `index.html` no navegador, ou publique via **GitHub Pages** (veja abaixo).

## Publicar no GitHub Pages (grátis)

1. Crie um repositório no GitHub (ex: `pirataria-2026`)
2. Faça upload de todos os arquivos desta pasta
3. Vá em **Settings → Pages**
4. Em "Branch", selecione `main` e pasta `/ (root)`
5. Clique em **Save**
6. Seu site estará em: `https://SEU_USUARIO.github.io/pirataria-2026`

## Gerenciar lotes esgotados

Para marcar um lote como esgotado **sem editar código**:
- Acesse o site e clique **5 vezes no rodapé**
- Ou adicione `?admin` no final da URL
- Use os toggles para ativar/desativar cada lote
